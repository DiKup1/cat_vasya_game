Реализация классов для ролевой игры с пошаговой боевкой.
Проект выполнен на языке Python в соответствии с требованиями PEP-8.
Проект разделен на 4 файла:
 1.unit.py:
   а Абстрактный базовый класс `Unit`.
   б Содержит общие характеристики (сила, ловкость, телосложение и т.д.).
   в Реализует систему заклинаний (`add_spell`, `cast_spell`).

 2. character.py
    а Класс `Character` (наследник `Unit`).
    б Поддерживает три игровых класса через параметр `character_class`:
      1 Warrior (Воин): Высокий урон и защита, малый запас маны.
      2 Mage (Маг): Высокий магический урон, большой запас маны.
      3 Hunter (Охотник):** Баланс характеристик, урон зависит от ловкости.
    в Характеристики рассчитываются автоматически при создании объекта.

 3 monster.py
    а Класс `Monster` (наследник `Unit`).
    б Реализует свои формулы расчета здоровья, урона и защиты.

4   spell.py
    а Абстрактный класс `Spell`.
    б Конкретные реализации заклинаний:
     1 Fireball (Огненный шар)
     2 IceLance (Ледяное копье)
     3 LightningBolt (Молния)



# Добавляем заклинание
mage.add_spell(Fireball())

# Используем заклинание
damage = mage.cast_spell(0)
print(f"Маг нанес {damage} урона!")
